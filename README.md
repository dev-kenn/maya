# Flutter Wallet App

A Flutter Money application using BLoC state management.

## Features

- User login authentication
- Wallet balance display
- Send money
- Transaction history
- API integration

## Tech Stack

- Flutter
- Dart
- flutter_bloc
- Repository Pattern
- REST API


## Installation Guide
## Prerequisites

Before running this project, make sure you have:

Flutter SDK installed
Dart SDK installed
Android Studio / VS Code
Xcode
Android Emulator or iOS Simulator

## Step 1: Clone the repository
git clone https://github.com/dev-kenn/maya.git

## Step 2: Go to project directory
cd maya

## Step 3: Install dependencies
flutter pub get

## Step 4: Run the project
flutter run
